# Feedback-Enhanced-Data-Synthesis

This Repo Contains The codes for Devoloping Human in the Loop Tabular Data Synthesis.

EDA_and_Prprocessing file contains the Intitial main dataset loading , Explorary data Analysis and relavant preprocessing for the data.

Main_final_code.ipynb contains the full code to Build the architecture, Load preprocessed data, And ttrain the proposed System, and the evaluation and Benchmarking. And it contains the code for testing the generalizability of the system against Two other datasets.

UI_code.ipynb contains the testing with a User interface for the system.
